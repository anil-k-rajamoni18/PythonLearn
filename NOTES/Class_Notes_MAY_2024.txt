 COMMANDS 
 ------------------
 WINDOWS 
	 dir 							- display the current path 
	 cd dirName/foldername/path 	- to change the path 
	 mkdir dirName 					- to create new dir or folder
	 rmdir dirName 					- to remove the dir or folder
	 cd .. 							- backward path change 
	 notepad filename.extension 	- creates new file 
	 type filename 					- display the content of the file
	 
 MAC-OS 
 ------------
	ls , ls -a , ls -ltr 			- display the list contents
	pwd								- display current curren path 
	cd 	dirName						- change the path
	mkdir dirName 					- create a new floder/dir
	rmdir dirName 					- delete existing empty dir/floder
	rm -R dir/floder 				- delete existing non empty dir/floder
	touch filename.ext 				- creates new file.
	cat filename					- display the content of the file.
	open filename 					- opens the file
